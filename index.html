Proyectos

Mascotas Perdidas

Django GitHub

Somos parte de una comunidad muy cercana a los animales, y como miembros de esta comunidad, cada uno de nosotros vivió o presenció la pérdida de una mascota.
De esta forma decidimos crear MascotasPerdidas.com.uy, un sitio web destinado a todos los usuarios que necesiten publicar anuncios de mascotas Perdidas o Encontradas.
Creemos que la mejor manera de ayudar a la comunidad es centralizando las publicaciones en un mismo sitio, a modo que, cada miembro que quiera colaborar con la búsqueda de una mascota, pueda hacerlo de una manera rápida y cómoda.

Hola Suerte!

Python GitHub

Aplicación para Sistemas operativos de PC, permite a los usuarios redactar un listado de participantes y la aplicación devolverá uno de forma aleatoria.
Sencilla pero con usos tan amplios como: Decidir que miembro de la familia lavará los platos esta noche o, en una reunión de amigos decidir quién va a invitar un trago más (de forma aleatoria). Los usos son ilimitados y recordar, el ganador (o perdedor) es sólo uno.

ImageLed Ride

Arduino GitHub

Sistema montado en Arduino UNO que genera una imagen o pequeño GIF en una o ambas ruedas de una bicicleta (en movimiento).
Mediante un sensor hall recolectamos el tiempo en el que la rueda da un giro completo, y durante ese tiempo reproducimos una matriz en una tira de leds ensamblada a mano por todo el radio de la rueda.
